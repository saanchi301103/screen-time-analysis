# screen-time-analysis

Project Overview
This project analyzes the impact of screen time on productivity and mental health among university students using a data-driven and machine learning approach.

Objectives
* Study relationship between screen time and productivity
* Analyze impact on stress and sleep
* Build prediction model using linear regression

Dataset
150 student records including:

* Screen Time
* Productivity
* Stress
* Sleep Quality

Tools Used
* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

Process
* Data cleaning
* Exploratory Data Analysis
* Visualization
* Linear Regression model

Results
* Screen time ↑ → Productivity ↓
* Screen time ↑ → Stress ↑
* Screen time ↑ → Sleep quality ↓
